
학생 정보 관리 프로그램 보고서

본 프로그램은 학생의 학번, 이름, 학과, 이메일, 전화번호 등의 정보를 관리하기 위한 C언어 프로그램입니다.
수업시간에 배운 파일 입출력 기능을 활용하여 학생 정보를 추가하고 검색하며, 특정 조건에 맞는 정보를 내보내거나 삭제할 수 있도록 구현되었습니다.
또한, 사용자의 잘못된 입력에 대한 예외 처리를 신경쓰어 사용자 친화적인 인터페이스를 구현하는데에 힘을 썼습니당.

※본 프로그램은 main.exe 로 실행해주시기 바랍니다. (한글이 깨질 우려가 있음)※

### 주요 기능 설명

#### 1. 학생 정보 추가 (`add_student` 함수)
- `add_student` 함수는 사용자로부터 학번, 이름, 학과, 이메일, 전화번호 정보를 입력받아 "학생정보파일.txt" 파일에 저장합니다.
- 파일은 "append" 모드로 열리므로 기존 파일 내용에 새로운 학생 정보가 추가됩니다.
- 입력된 정보는 CSV 형식(데이터과학시간에 배운 쉼표 구분 , )으로 파일에 저장됩니다.

#### 2. 학생 정보 검색 (`search_student` 함수)
- 사용자가 학번 또는 학과로 검색을 선택할 수 있으며, `scanf`를 통해 잘못된 입력이 들어온 경우 예외 처리가 됩니다. (ex int search_type;)
- 숫자가 아닌 입력이나, 유효하지 않은 선택(1 또는 2가 아닌 경우)에 대해 "잘못된 입력입니다. 1 또는 2를 선택해 주세요." 메시지를 출력하고 다시 입력받습니다.
- 일치하는 학생 정보를 찾을 경우 해당 정보를 화면에 출력합니다. 일치하는 정보가 없으면 "검색 결과가 없습니다." 메시지를 표시합니다.

#### 3. 학생 정보 내보내기 (`export_students` 함수)
- 전체 학생 정보를 내보내거나 특정 학번 또는 특정 학과의 학생 정보를 별도 파일로 저장할 수 있습니다.
- 내보내기 방식 선택 시, 특정 학번의 경우 "{학번}_학생정보.txt"라는 이름으로, 특정 학과의 경우 "{학과}_학생정보.txt"로 파일이 저장됩니다.

#### 4. 학생 정보 삭제 (`delete_student` 함수)
- 특정 학번의 학생 정보를 삭제할 수 있습니다.
- 삭제할 학생 정보를 제외한 모든 데이터를 임시 파일에 저장한 후, 원본 파일을 삭제하고 임시 파일을 원본 파일명으로 변경하여 삭제된 내용만 남도록 합니다.

#### 5. 메뉴 선택과 예외 처리 (`main` 함수)
- `main` 함수에서 `scanf`를 통해 메뉴 선택 시에도 잘못된 입력을 방지합니다.
- 숫자가 아닌 값을 입력할 경우 "잘못된 입력입니다. 다시 선택해 주세요." 메시지가 표시되며, 입력 버퍼를 비우고 올바른 입력이 들어올 때까지 반복합니다.
